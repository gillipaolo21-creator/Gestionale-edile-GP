# Gestionale Edile GP

Repo principale per backend (NestJS) e frontend (Next.js).

## Avvio locale

1. Crea i file env partendo dagli esempi.
2. Installa le dipendenze.
3. Avvia in sviluppo.

```bash
pnpm install
pnpm dev
```

## Docker (sviluppo)

Per usare Docker in sviluppo, segui la guida in [DOCKER_DEV.md](DOCKER_DEV.md).
